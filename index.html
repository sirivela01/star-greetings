Database Settings

To play online and use social login, you need a Firebase project. Enter your settings below:

Step-by-step setup (1 minute):
1. Go to Firebase Console and click Add Project.
2. Build > Realtime Database > click Create Database.
3. In the Rules tab, set read/write to true:
{
  "rules": {
    ".read": true,
    ".write": true
  }
}
4. Copy your Database URL from the Data tab and paste it above!
